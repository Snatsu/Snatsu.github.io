我想要钱

知识点：php的科学记数法，time()函数

打开题目，一道代码审计：
time()函数返回当前的unix时间戳。
money应该少于等于四位且大于当前的unix时间戳。那么它大概有多大呢？
将当前时间转换为时间戳：1555848362，是个十位的大数。 要让最多四位的money大于这个数，就要用到科学记数法。

payload：

?money=1e10

得到flag。

关于科学计数法

P.S. 虽然在excel中1.00E+10是正确写法，但1e10好像也并非是错误的。

文件包含

知识点：文件包含，phpfilter协议流

打开题目，url为说明可利用文件包含。在源代码注释中发现有

<!--flag.php-->

直接访问没有发现flag。

于是使用filter协议，构造payload:

?file=php://filter/read=convert.base64-encode/resource=flag.php

得到一串base64：

SSBoYXZlIGEgZmxhZyEKPD9waHAgCgovL0ZsYWc6IG1vY3Rme2YxbGVfaW5jbHVkNF9lNXN5fQovL0J5OmRhb3l1YW4KCj8+Cg==

解码得到flag。

P.S. base64编码前为php代码会被直接执行，而编码后可以读取。

知识点：base64绕过php exit

打开题目，发现直接给出一段代码。


可以看到$c初始值为”<?php exit; ?>”，然后.=我们post进去的c值。

即：

<?php
    exit;
?>
xxxxx

其中xxxxx是我们传进去的数据。

所以无论我们传入什么数据，$c的头部都会是一段exit，直接退出脚本。那么，现在的问题就是如何绕过这段exit了。

经百度得知，可以使用phpfilter协议流进行绕过。

于是构造payload

c=aPD9waHAgc3lzdGVtKCdjYXQgZmxhZy5waHAnKTsgPz4=&file=php://filter/write=convert.base64-decode/resource=tmp.php

利用hackbar，post，成功绕过。

unset

知识点：php unset函数，文件包含，伪协议

前言

花了很长时间搞懂这道题的思路，是一道很有意思的代码审计题目

正文

首先打开题目，得到源代码。

可以看到先是定义了一个waf函数，然后foreach进行了一个操作，然后是主要部分

一块块地来审计。

首先

function waf($a){
    foreach($a as $key => $value){
        if(preg_match('/flag/i',$key)){
            exit('are you a hacker');
}
}
}

可以得知waf的功能为检测传入的变量中的键值对的键是否含有’flag’，若含有，则直接退出并打印出’are you a hacker’。

另外，这里正则表达式里的修饰符’i’的功能是使匹配对大小写不敏感，刚开始看到时还以为能够利用，不过这道题目没有用到就是了。

然后 

foreach(array('_POST', '_GET', '_COOKIE') as $__R) {
    if($$__R) { 
    foreach($$__R as $__k => $__v) { 
        if(isset($$__k) && $$__k == $__v) unset($$__k); 
    }
 }
}

这一部分也是最难理解的一部分，姑且把自己的理解写下来，望大佬勿喷。

简单来说，先将_POST,_GET,_COOKIE三个全局变量作为一个数组赋值给__R，然后如果任一全局变量(数组)中存在变量，则遍历变量作为键值对，且若键的值存在且键的值等于值的值(有点绕。。)，则使用unset函数销毁掉键的值这一变量。

往下走，

if($_POST) { waf($_POST);}
if($_GET) { waf($_GET); }
if($_COOKIE) { waf($_COOKIE);}

分别对_POST,_GET,_COOKIE调用三次waf。

然后

if($_POST) extract($_POST, EXTR_SKIP);
if($_GET) extract($_GET, EXTR_SKIP);

可以通过extract函数传入变量的值。注意：此处的extract函数中第二个参数为’EXTR_SKIP’，即当同名变量存在时会跳过，并没有变量覆盖漏洞。

最后

if(isset($_GET['flag'])){
if($_GET['flag'] === $_GET['daiker']){
        exit('error');
}
if(md5($_GET['flag'] ) == md5($_GET['daiker'])){
        include($_GET['file']);
}
}

简单的md5弱类型比较。
