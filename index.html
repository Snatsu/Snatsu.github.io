知识点：302跳转，条件竞争

打开题目链接，跳转到index2.php。考虑到一般都是直接index.php，所以尝试一下，发现还是会跳转到index2。猜测可能是302跳转。

burp抓包。action->Send to Repeater将index2改为index，Go。

访问uploadsomething.php
试着随便传了下文件，得知要得到flag就要访问给出的文件路径。
访问路径，却显示“Too Slow！”

这里卡了好久，后来想到应该是上传后立刻服务器将文件删除。此处就可以利用条件竞争漏洞。

  • 使用burp不停上传文件
  • 使用脚本不停访问文件
    上传如图内容取得路径：http://119.23.73.3:5006/web2/uploads/5a3e88b9cef15e18d7d6f538b48769237bb12f2e/233

    burp

    burp抓包 action->send to intruder
  • clear $
  • payload type:null payloads
  • payload options -> continue indefinitely
  • number of threads:20

    py脚本

    脚本如下：
import requests
u = "http://119.23.73.3:5006/web2/uploads/5a3e88b9cef15e18d7d6f538b48769237bb12f2e/233"
t = requests.get(u).text
while True:
    print(t)

得到flag。
